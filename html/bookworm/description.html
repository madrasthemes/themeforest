Bookworm HTML is a high-quality and well coded HTML Bootstrap Template specially designed to fit all the needs of a Book Store. This HTML template including book list, book detail, author page, shop pages and also blog. Bookworm HTML is a very complete and highly customizable HTML template that will help you build your own Bookstore layouts.

This template includes 56 carefully built using Bootstrap 4, which are easy to edit and customize to fit your needs. Template is perfect solutions for the create of unique web projects.

Bookworm: Simple, elegant, extensive, professional.
